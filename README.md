# Aw Sheng You — Portfolio

Responsive portfolio for a Singapore-based digital designer and developer, built with Next.js, TypeScript, React Three Fiber, and Motion.

## Local development

```bash
npm install
npm run dev
```

Quality checks: `npm run lint`, `npm run typecheck`, `npm test`, and `npm run build`.

## Replaceable assets

Add the final resume under `public/documents`, project media under `public/images` or `public/video`, and certificate previews under `public/certificates`. Cooper Black and any third-party character asset must only be added with a valid web license. The current procedural mascot is original and intentionally temporary.

Set `NEXT_PUBLIC_SITE_URL` to the production Vercel URL before release.
